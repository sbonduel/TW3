# Installation de NPM
<Callout type='info' emoji="⚠️">
*Les postes de l’université sont déjà configurés avec NPM.*
</Callout>

NPM (Node Package Manager) est installé automatiquement avec Node.js, le runtime JavaScript qui permet d'exécuter du code JavaScript en dehors d'un navigateur. Node.js et NPM sont tous deux essentiels pour gérer les dépendances et les outils utilisés dans les projets JavaScript modernes. 

**Node.js** fournit l'environnement d'exécution, tandis que **NPM** permet de gérer les bibliothèques (ou "paquets") dont un projet a besoin pour fonctionner. Avant de commencer à utiliser NPM, il est crucial de s'assurer que Node.js est bien installé, car NPM est directement lié à cette installation.

## Étapes d’installation :
1. **Télécharger Node.js** : Pour installer NPM, vous devez d'abord télécharger et installer Node.js depuis le site officiel [nodejs.org](https://nodejs.org). Vous avez généralement le choix entre deux versions :
   - **LTS (Long-Term Support)** : version stable recommandée pour la majorité des projets.
   - **Current** : version avec les dernières fonctionnalités, plus adaptée aux développeurs souhaitant expérimenter.

2. **Vérification de l’installation** : Une fois Node.js installé, NPM est automatiquement inclus. Vous pouvez vérifier que tout est bien installé en exécutant les commandes suivantes dans votre terminal :
   ```bash copy
   node -v   # Affiche la version de Node.js
   npm -v    # Affiche la version de NPM
   ```

3. **Mise à jour de NPM** : Il est parfois nécessaire de mettre à jour NPM après l'installation de Node.js. Pour cela, utilisez la commande suivante :
   ```bash copy
   npm install -g npm
   ```

Avec ces étapes simples, vous serez prêt à utiliser NPM pour gérer vos projets, installer des paquets et automatiser des tâches de développement.
