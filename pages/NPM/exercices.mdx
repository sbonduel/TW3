# Exercices pratiques

## Initialisation d’un projet NPM
1. Créez un nouveau dossier pour un projet NPM.
2. Exécutez `npm init` pour créer un fichier `package.json`.
3. Installez un paquet simple, comme `express` :
   ```bash copy
   npm install express
   ```
4. Vérifiez que le paquet est correctement listé dans `package.json`.
## Utilisation des scripts NPM
1. Ajoutez un script `start` dans `package.json` pour démarrer un serveur Express.
2. Testez l’exécution du script avec `npm start`.
## Gestion des dépendances
1. Installez des paquets de développement comme `nodemon` pour le redémarrage automatique du serveur lors des modifications de fichiers :
   ```bash copy
   npm install --save-dev nodemon
   ```
2. Utilisez `nodemon` pour démarrer votre serveur avec un script personnalisé.
